Sensor Suite

Driven by the requirements for high-speed maneuvers, our sensor suite comprises only a multi-camera setup, with inertial data provided by the built-in IMU of the event cameras. The specific sensor models and their parameters are shown in the table below:

Sensor Description
Event Cameras 2 x DVXplorer
640×480 resolution, 10cm baseline
Frame-based Camera Intel RealSense D435
2 × 640×480 Grayscale, 5cm baseline, 30Hz
1 × 640×480 RGB, 30Hz
IMU Bosch BMI-160
800Hz sampling rate
Motion Capture Device LUSTER FZMotion - 120Hz (for drone)
NOKOV Mars26H - 60Hz (for others)
